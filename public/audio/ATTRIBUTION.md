# Futbol audio attribution

The runtime supports two preloaded recorded-audio files:

- `stadium-ambience.ogg`
- `stadium-reaction.ogg`

No crowd recording is bundled in this revision. The automated browser session
blocked media downloads from both researched CC0 sources, so the game disables
the recorded crowd layer cleanly instead of falling back to generated white
noise or unlicensed audio.

Researched compatible sources:

1. **Crowd Shouting/Speaking Ambience**
   - Creator: StarNinjas
   - Source: https://opengameart.org/content/crowd-shoutingspeaking-ambience
   - License: CC0
   - Original filename: `crowd_shouting_0.ogg`
   - Download date: not downloaded
   - Modifications: none
2. **Sound Effects - Applause after a concert.ogg**
   - Creator: Amada44
   - Source: https://commons.wikimedia.org/wiki/File:Sound_Effects_-_Applause_after_a_concert.ogg
   - License: CC0 1.0
   - Original filename: `Sound_Effects_-_Applause_after_a_concert.ogg`
   - Download date: not downloaded
   - Modifications: none

Browser `speechSynthesis` commentary and synthetic crowd-noise generation are
disabled. Commentary is intentionally omitted until redistributable recorded
clips are supplied.
